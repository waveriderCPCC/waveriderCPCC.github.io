what's my brand?

I decided to make this website's design brand myself.
wave/waverider is my username; and fits in the header well.
It's a pretty simple design; it's job is to enhance the delivery of the information on here.
and also just to look nice.

some styles that this site is inspired by (but not any one in particular):

  • Frutiger Aero
  • Skeumorphism
  • Windows 11 / stained glass

tagline

"Clearance is a Boolean"

header style

the title (the one that says waverider • <page>, uses Nunito, a drop shadow, and a subtle blue stroke that shows through to the background (color #4), besides also being filled with black:

        
        /* waverider • home */
        h1 {
          text-align: center;

          font-family: "Nunito", sans-serif;
          font-style: normal;
          font-weight: 700;
          font-size: 50px;
          line-height: 44px;
      
          color: #000000;
      
          text-shadow: 0px 4px 4px rgba(0, 0, 0, 0.25);
          paint-order: stroke fill;
          -webkit-text-stroke: 4px rgba(152, 213, 230, 0.2);
          -webkit-text-fill-color: black;
        }

the top nav bar uses this styling, as well as the font Poppins:

      
        .topmenu {
          color: black;
          text-align: center;
          font-family: "Poppins", sans-serif;
          font-weight: 400;
          font-size: 18px;
          text-shadow: 0px 0px 4px rgba(0, 0, 0, 0.25);
        }
